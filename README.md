## 개요
  게임 타이틀: ERROR\
  장르: TPS\
  플랫폼: PC\
  개발: Unity 3D
  
## 소개
세계 최고의 특수부대인 ERROR팀에 주인공이 입대하게 되면서 수행하는 임무들을 직접 플레이하는 게임입니다.\
3인칭 슈팅게임이며 각 스테이지의 보스를 잡는 것이 목표입니다. 튜토리얼과 임무 스테이지 2개로 이루어져 있습니다.

## 구현
- 캐릭터 조작
- 스킬 시스템 구현
- 무기 시스템 구현
- 적마다 다른 공격 패턴
- 보스 페이즈 시스템
- 외부 파일 데이터로드
